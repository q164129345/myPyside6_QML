┌──────────────────────────────────────────────────────────────┐
│ 🚀 快速开始                                                   │
└──────────────────────────────────────────────────────────────┘

  1. 启动程序:
     
     python main_loader.py
  
  2. 编辑 QML:
     
     打开: Example/Main_content.qml
     修改任何内容
  
  3. 保存查看:
     
     Ctrl+S → 右上角显示"✅ 已重载" → UI 丝滑更新!

┌──────────────────────────────────────────────────────────────┐
│ 📁 文件说明                                                   │
└──────────────────────────────────────────────────────────────┘

  ✅ main_loader.py        → 主程序 (运行这个)
  ✅ Main_content.qml      → 你的 UI (编辑这个)
  ⚙️  Main_wrapper.qml      → 包装器 (自动生成,不要动)

┌──────────────────────────────────────────────────────────────┐
│ 🎯 工作原理                                                   │
└──────────────────────────────────────────────────────────────┘

  ApplicationWindow (固定)
      └── Loader (动态)
              └── Main_content.qml (你的内容)
  
  关键: 通过改变 Loader.source URL (加时间戳) 强制重新加载

┌──────────────────────────────────────────────────────────────┐
│ ❓ 常见问题                                                   │
└──────────────────────────────────────────────────────────────┘

  Q: 修改了但没更新?
  A: 检查是否编辑的是 Main_content.qml (不是 Main_wrapper.qml)
  
  Q: 报语法错误?
  A: 查看控制台错误信息,修复后重新保存
  
  Q: 想修改 Python 代码?
  A: 需要重启整个程序 (Ctrl+C 然后重新运行)
